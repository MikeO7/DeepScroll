## Installation

1. Clone the repository.
2. Run `npm install`.
3. Run `npm run build`.
4. Load the `dist/` directory as an unpacked extension in Chrome/Edge.

## Usage

1. Click the DeepScroll icon or use `Alt+Shift+P`.
2. Click "Capture Page".
3. Wait for the scrolling to complete.
4. Edit and save your capture!
